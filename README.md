## 内容说明

- 章节内容已经翻译为中文并保留原有公式结构，适合直接在线查阅
- docsify 入口文件是 [index.html](index.html)，不需要额外构建
- 翻译脚本位于 [scripts/translate_mlsys.py](scripts/translate_mlsys.py)，原理是调取Google Translate的API，所以可能有一些typo问题，欢迎纠错。
