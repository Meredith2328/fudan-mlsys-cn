# MLSys 中文整理

这个仓库把以下两个网页整理成了适合 docsify 展示的中文 Markdown，直接推送到 GitHub 仓库后即可开启 GitHub Pages：

- [第 1 章：机器学习系统：问题、约束与垂直整合](chapter1.md)
- [第 2 章：CPU 基础、GPU 的兴起与吞吐计算的逻辑](chapter2.md)

## 内容说明

- 原始来源分别是 [Chapter 1](https://memxlife.github.io/books/mlsys/chapter1.html) 和 [Chapter 2](https://memxlife.github.io/books/mlsys/chapter2.html)
- 章节内容已经翻译为中文并保留原有公式结构，适合直接在线查阅
- docsify 入口文件是 [index.html](index.html)，不需要额外构建
- 翻译脚本位于 [scripts/translate_mlsys.py](scripts/translate_mlsys.py)

## 使用方式

1. 把当前目录推送到 GitHub 仓库。
2. 在仓库设置中开启 GitHub Pages。
3. 选择从 `main` 分支的根目录发布。
